# Linguagem REXX

### Operadores aritméticos

- Adição
  
      + 

- Subtração
      
      -

- Multiplicação
  
      *
  
- Divisão

      /
      
- Divisão de número inteiro (divida e retorne a parte inteira do resultado)

      %
      
- Resto da divisão

      //
      
- Potência

      **
      
- Subtração com prefix (Igual a subtração)

      Prefix -

- Adição com Prefix (igual a adição)

      Prefix +
